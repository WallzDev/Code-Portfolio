# Fior Eterna (Professional experience)

**Developer Notes** 
Due to certain agreements it is not possible to showcase the game, but in this repository there are some scripts showing little of what was worked on during this project.

### Project Overview
Fior Eterna is a single player 3D puzzle game yet to be released.

### Technical Implementation (Team Development)
This project had a small team, as lead of programming and development i worked on most main mechanics and puzzles, as well as implemented the assets and assembled most of the main game.
There was also additional work done on the Save System and other actions given by the tool GameCreator2 so that the team could blend the tool and the regular C# scripts.
